<objective>
Rename the two top-level generator pipeline types — `FluentFactoryGenerator` (the `IIncrementalGenerator` entry point) and `FluentFactoryCompilationUnit` (the per-root compilation output record) — to `FluentRootGenerator` and `FluentRootCompilationUnit`. Move their source files via `git mv` so `git log --follow` preserves the history. Update every call site inside `src/Converj.Generator/` and every test reference inside `src/Converj.Generator.Tests/` (the test verifier is parameterized on `FluentFactoryGenerator`, so the rename ripples across ~50 test files).

Closes NAME-01, NAME-02, and the FILE-01 requirement for these two files.

Purpose: Align the most-visible internal pipeline types with the public `[FluentRoot]` vocabulary shipped in v2.0. These two types are the conceptual "front door" of the generator and their legacy `Factory` naming is the highest-priority drift in the `Converj.Generator` project.

Output: Two renamed `.cs` files (with preserved git history), updated call sites in 4 generator files plus all matching test files, green build, green test suite.
</objective>
