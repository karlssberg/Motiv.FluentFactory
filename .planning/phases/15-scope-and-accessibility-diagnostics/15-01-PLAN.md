<objective>
Implement and test two new diagnostic rules: MFFG0012 for private/protected constructors marked with [FluentConstructor], and MFFG0013 for factory root types missing the `partial` modifier.

Purpose: These diagnostics prevent users from silently misconfiguring their fluent factories with inaccessible constructors or non-partial types that cannot have generated code added.
Output: Two new diagnostic descriptors, validation logic, and passing TDD tests.
</objective>
