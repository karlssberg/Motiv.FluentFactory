       - `var attr = parameter.GetAttributes(TypeName.FluentCollectionMethodAttribute).FirstOrDefault();` (use existing `SymbolAttributeExtensions.GetAttributes(ISymbol, TypeName)`). Skip parameter if null.

           ```csharp
           private static string? TryDeriveAccumulatorName(string paramName)

               var singular = paramName.Singularize();
               if (string.IsNullOrWhiteSpace(singular)) return null;
               if (singular == paramName) return null;
               if (SyntaxFacts.GetKeywordKind(singular) != SyntaxKind.None) return null;

           }
           ```
