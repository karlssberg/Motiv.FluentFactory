<!-- FluentMethodParameter equality drives Trie merging -->
From FluentMethodParameter.cs:
```csharp
// Two FluentMethodParameters are equal if they have the same FluentType AND overlapping Names
// GetHashCode uses only FluentType -- so same-type params always collide in hash
public bool Equals(FluentMethodParameter? other)
{
    if (!FluentType.Equals(other.FluentType)) return false;
    return Names.Overlaps(other.Names);
}
public override int GetHashCode() => FluentType.GetHashCode();
```

Generated code always:

- Adds [GeneratedCode] attribute

- File named `{Namespace}.{FactoryTypeName}.g.cs`
